# JaegerBot

A Discord Bot with generic utility and Tarkov knowledge

- `roll`                    prints 1-6 randomly
- `coinflip/coin`           prints Heads or Tails
- `troll string_to_troll`   JUST loOk aT ThIs

- `about/source/code`       Developer information
